Sobre Mí

Desarrollador Full Stack con más de 4 años de experiencia en el desarrollo de soluciones robustas y escalables. Con sólidos conocimientos en fundamentos de ingeniería de software, aplicando arquitecturas limpias (Clean Arquitecture, DDD, y N-Capas), principios SOLID y Clean Code para garantizar código mantenible y extensible. Conocimiento en metodologías ágiles (Scrum, XP, Mobile-D) y en todas las fases del ciclo de vida del desarrollo de software (SDLC), desde el análisis de requerimientos hasta el despliegue, siempre aplicando KISS y YAGNI para evitar la sobre-ingeniería.


Mi enfoque se centra en la calidad, seguridad y optimización de performance, utilizando tecnologías como .NET, Spring Boot, Laravel, AngularJS, MS SQL Server, PostgreSQL, Cloud (AWS, GCP), DevOps (CI/CD con GitHub Actions). Implementando patrones de diseño, inyección de dependencias y buenas prácticas para asegurar sistemas eficientes y confiables.


Mi objetivo: desarrollar software escalable, seguro y de alto rendimiento, combinando las mejores prácticas de la industria con tecnologías modernas para entregar soluciones innovadoras y sostenibles.
